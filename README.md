# MaidMatch Website

A modern, responsive website for the MaidMatch mobile application. This website showcases the app's features, services, and coverage areas.

## Features

- Responsive design that works on all devices
- Modern UI with smooth animations
- Brand-consistent purple color scheme
- Easy navigation with smooth scrolling
- Social media integration

## Structure

- `index.html` - Main webpage content
- `styles.css` - All styling and animations
- `script.js` - Interactive elements and animations
- `assets/` - Directory containing images and icons

## Setup

1. Clone this repository
2. Add your images to the `assets` directory:
   - logo.png
   - hero-app.png
   - google-play.png
   - app-store.png

## Development

To run this website locally, simply open `index.html` in your web browser.

## Brand Colors

- Primary Color: #6B2C91 (Purple)
- Secondary Color: #F06292 (Pink)
